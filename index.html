REFRAGMENTOR


An image (de)fragmentor.

INSTRUCTIONS:

1. Take or obtain an image of something meaningful to you.

2. Upload your image to this website. A new image will be automatically generated. (Nothing will be saved or collected, this program runs locally).

3. Save and do whatever you'd like with the image.

Some suggestions include:
- Print it out and use it like a talisman.
- Make up a ritual and burn it.
- Delete it.
- Use it in a collage.
- Refragment the refragmented image.

Only you can see this image. It's yours now.

Optional: You can also share it with Aldon if you'd like to be included in the website gallery and/or cute little slideshow above.
